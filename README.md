## Description

peed Unit Changer - Simple plugin to visualize current speed in different units.

Displays speed in different units in Navball.
Current speed units displayed:
-Meters per second
-Kilometers per hour
-Miles per hour
-Knots
-Feet per second
-Mach

Now it also displays a window with the altitude depending on the unit selected for altitude:
- Meters
- Kilometers
- Miles
- Nautical miles
- feet
